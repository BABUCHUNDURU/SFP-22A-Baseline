FasStudentAwardInformationV2Event (Outbound)

awardingoutbound
Message Class
FasStudentAwardInformationV2Event
Description

Student award information for one academic year

XML Schema

FasStudentAwardInformationV2.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action
  • A fund status becomes Awarded
  • A fund's "Maximum Projected Award Amount up to Packaging Limit" increases
