Oracle SFP Messaging API 22.1.0.0

FasStudentNotificationEvent (Outbound)

studentnotificationoutbound
Message Class
FasStudentNotificationEvent
Description

Message used to notify students of actions they must take to receive financial aid and provide them with informative messages as to their financial aid status.

XML Schema

FasStudentNotification.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.
