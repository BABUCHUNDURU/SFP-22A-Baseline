FasUnmatchedIsirStudentRecordEvent (Outbound)

outboundisir
Message Class
FasUnmatchedIsirStudentRecordEvent
Description

This message defines the data SFP will send to a destination system for students with unmatched ISIR information that can be used for lead management processing.

XML Schema

FasUnmatchedIsirStudentRecordEvent.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action
  • The user has selected that there are no student matches in the additional ISIR matching task and
  • the school has configured the system to send student record information from the unmatched ISIR to an external system (e.g. CRM or other lead management system) and
  • it has met the configured amount of days since the ISIR receipt date
