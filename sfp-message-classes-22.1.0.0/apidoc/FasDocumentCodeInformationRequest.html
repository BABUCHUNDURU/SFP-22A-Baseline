FasDocumentCodeInformationRequest (Request/Reply)

webservicedocumentsync
Request Message Class
FasDocumentCodeInformationRequest
Reply Message Class
FasDocumentCodeInformationReply
Description

Service that returns document code configuration data.

XML Schema

FasDocumentCodeDetailInformation.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/in.sync/publish-consume
Message Exchange Pattern

Synchronous inbound request-reply

A client system sends a message in the HTTP request and waits until the reply message is returned in the HTTP response. Please see "MPG Public API for Developers" for additional details.

By design, this mode of communication always operates in read-only mode. This means that it can only be used to retrieve data that is already available in SFP.

Request Details

Contains the following fields:

  • DocumentCode (optional)
  • AwardYear (optional)

If no DocumentCode or AwardYear are provided, all document code definitions will be returned. If only the DocumentCode is provided the system will return all document codes that match the provided DocumentCode regardless of AwardYear. If only the AwardYear is provided the system will return document codes that match the given AwardYear.

Reply Details

Document Code details.

Error Handling

If the request message contains validation errors, a MessageValidationErrorReply message will be returned in the HTTP response. Please check the payload of this message for details about the error.
