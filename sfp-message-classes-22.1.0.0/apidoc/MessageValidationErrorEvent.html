MessageValidationErrorEvent (Outbound)

errorhandlingoutboundvalidation
Message Class
MessageValidationErrorEvent
Description

Message published by SFP when an inbound message contains validation errors.

XML Schema

common/VocadoCommon.xsd

XML Namespace
http://www.vocado.com/vm/common
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action

A received message (from Client to SFP) contains validation errors.

Notes:

  • A message is considered invalid when:
    • the XML document does not match the XML schema
    • a business rule is violated
  • Validation rules run before SFP performs any additional processing.
  • See also MessageValidationErrorReply for validation errors in a request-reply scenario.
