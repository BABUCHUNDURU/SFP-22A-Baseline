FasLetterNotificationEvent (Outbound)

lettersoutbound
Message Class
FasLetterNotificationEvent
Description

Message used to notify that a letter has been generated.

XML Schema

FasLetter.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action
  • Triggering action defined for the letter occurs in the FAS
  • User requests to manually generate a previously created letter in the FAS
  • SFP will trigger the outbound message with only the defined data parameters included in the outbound message
