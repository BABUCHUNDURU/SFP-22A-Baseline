Oracle SFP Messaging API 22.1.0.0

FasDocumentRejectedEvent (Outbound)

outbounddocument
Message Class
FasDocumentRejectedEvent
Description

Event message that informs that a document was rejected.

XML Schema

FasDocumentRejectedEvent.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action

SFP has identified that a document has been rejected, either due to an "unacceptable" DMS status or manually rejected from the FAS.
