Oracle SFP Messaging API 22.1.0.0

FasLetterInformationRequest (Request/Reply)

webserviceletterssync
Request Message Class
FasLetterInformationRequest
Reply Message Class
FasLetterInformationReply
Description

Service that returns letter PDFs for a given student.

XML Schema

FasLetter.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/in.sync/publish-consume
Message Exchange Pattern

Synchronous inbound request-reply

A client system sends a message in the HTTP request and waits until the reply message is returned in the HTTP response. Please see "MPG Public API for Developers" for additional details.

By design, this mode of communication always operates in read-only mode. This means that it can only be used to retrieve data that is already available in SFP.

Request Details

Contains the following fields:

  • Letter ID (required)
Reply Details

A matching Letter PDF will be returned.

Error Handling

If the request message contains validation errors, a MessageValidationErrorReply message will be returned in the HTTP response. Please check the payload of this message for details about the error.
