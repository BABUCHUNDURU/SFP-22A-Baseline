FasIsirInformationRequest (Request/Reply)

webserviceisirsync
Request Message Class
FasIsirInformationRequest
Reply Message Class
FasIsirInformationReply
Description

Web service that returns ISIRs for a student and award year.

XML Schema

FasIsirInformation.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/in.sync/publish-consume
Message Exchange Pattern

Synchronous inbound request-reply

A client system sends a message in the HTTP request and waits until the reply message is returned in the HTTP response. Please see "MPG Public API for Developers" for additional details.

By design, this mode of communication always operates in read-only mode. This means that it can only be used to retrieve data that is already available in SFP.

Request Details

Contains the following fields:

  • Id (required);
  • SSN or ExternalStudentId;
  • list of (default: latest Award Year)
  • IsirsReturned (required): If all ISIRs or only the active one should be returned;
  • IsirFieldsReturned (required): Comma-separated list with the numbers of all ISIR fields to retrieve. It's also possible to request all fields by using the wildcard character "*". Note that field numbers might change per award year, according to current the EDE Tech Ref.
Reply Details

List of ISIR records.

Error Handling

If the request message contains validation errors, a MessageValidationErrorReply message will be returned in the HTTP response. Please check the payload of this message for details about the error.
