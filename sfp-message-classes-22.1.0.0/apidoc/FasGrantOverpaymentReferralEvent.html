Oracle SFP Messaging API 22.1.0.0

FasGrantOverpaymentReferralEvent (Outbound)

Message Class
FasGrantOverpaymentReferralEvent
Description

TThis message is sent to provide a view of a student's overdue Grant Overpayment funds.

XML Schema

FasGrantOverpayment.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action
  • Upon R2T4 re-trigger
  • Upon SAFI receipt
  • Upon expiration of the Grant Overpayment Resolution Timeframe (e.g., 45 days)
