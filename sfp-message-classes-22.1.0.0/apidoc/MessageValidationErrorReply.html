MessageValidationErrorReply

errorhandlingvalidation
Message Class
MessageValidationErrorReply
Description

This is a special reply message that is returned in the HTTP response when a client sends a synchronous message (message class ending with the Request suffix) that contains validation errors or cannot be processed.

When a request fails, MessageValidationErrorReply replaces the original reply message in order the provide details about the error.

Notes:

  • A message is considered invalid when the request
    • does not match the XML schema
    • violates one or more business rules
  • See also MessageValidationErrorEvent for validation errors in inbound events.
XML Schema

common/VocadoCommon.xsd

XML Namespace
