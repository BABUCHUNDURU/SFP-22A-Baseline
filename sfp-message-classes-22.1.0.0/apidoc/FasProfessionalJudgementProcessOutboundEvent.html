Oracle SFP Messaging API 22.1.0.0

FasProfessionalJudgementProcessOutboundEvent (Outbound)

professionaljudgementoutbound
Message Class
FasProfessionalJudgementProcessOutboundEvent
Description

This message is sent to notify students of actions they must take to receive financial aid and provide them with informative messages as to their financial aid status.

XML Schema

FasProfessionalJudgementInformation.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action
  • Professional Judgement process has been initiated
  • Professional Judgement process is on hold
  • Required document for the PJ process is received and is accepted or rejected
  • Additional supporting documents are requested within the PJ process
  • Professional Judgement process has been completed and an approval option has been made
