Oracle SFP Messaging API 22.1.0.0

FasFinancialPlanOutboundEvent (Outbound)

fpooutboundfinancialplan
Message Class
FasFinancialPlanOutboundEvent
Description

This message is sent to provide a complete view of a student's financial plan.

XML Schema

FasFinancialPlanOutbound.xsd

XML Namespace
http://www.vocado.com/vm/fas
HTTP Endpoint
/channels/out.ev/consume
Message Exchange Pattern

Outbound event message (from SFP to client).

SFP publishes an outbound message that can be consumed by a client system. Please see MPG documentation for additional details on how messages are consumed.

Trigger Action

There is a material change in the student's package:

  • Update to any Disbursement Amount
  • Update to any Disbursement Date
  • Fund/Disbursement is added
  • Fund/Disbursement is removed
