Student Financial Planning : STUDENT_AWARD_VW

Created by Stephen Hutchinson, last modified by Angshuman Mukherjee on Jun 15, 2021

This provides information for all student awards. This includes Awardable, Non-Awardable and Awarded Funds. The field enrollment_id to be used to link with STUDENT_CONSOLIDATED_VW. Linking columns are enrollment_id, fund_code, academic_year_number, payment_period_number
