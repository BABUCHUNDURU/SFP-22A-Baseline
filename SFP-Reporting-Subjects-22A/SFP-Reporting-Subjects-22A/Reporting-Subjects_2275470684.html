Student Financial Planning : Reporting Subjects

Created by Buck Foreman, last modified by Stephen Hutchinson on Jun 15, 2021

Oracle has released a new way to report against Student Financial Planning data. Oracle has developed a set of reporting views that consolidate common data sets into views to make querying related data easier. Oracle has also updated the view and column names to make it easier to find the data a report writer is looking to access.


This documentation will list the different views by subject areas and allow you to see how different views relate together as well as drill down into the detail definition of a view and see all the data fields contained within the view. In many cases the data has been consolidated so a reporting subject can be queried directly without having to relate another view to pull in additional information. This is expressed in the documentation with various subjects having a collection of reporting views that stand along and do not require being joined to other reporting views in the same subject area.


Not all of the reporting subjects maintain data in real time. The refresh schedule is listed on the SFP Reporting Subject Data Refresh page.
