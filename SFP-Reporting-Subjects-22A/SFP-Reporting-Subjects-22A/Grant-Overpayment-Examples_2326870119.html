Student Financial Planning : Grant Overpayment Examples

Created by Stephen Hutchinson, last modified on Oct 26, 2020

These are examples of queries that reference views in the Grant Overpayment subject.

The code convention $P{INPUT_PARAMETER} is single value input parameter and $X{IN, COLUMN_NAME, INPUT_PARAMETER} is the equivalent of a SQL Language "IN ()" statement. Both of these conventions are used specifically in Jasper Server and map to the input criteria defined within a report definition.

Grant Overpayment Report
SELECT a1.school_student_id
    -- , a.enrollment_id
     , a.fund_code
     , c.amount_to_return    as initial_amount
     , a.amount              as current_amount
     , CASE WHEN x.applied_amount IS NULL 
            THEN 0
            ELSE x.applied_amount
       END           as applied_amount    
     , (c.amount_to_return - a.amount)  total_amount   
     , a.gop_indicator            as status
     , a.date_of_determination
     , a.payment_due_date
     , CASE WHEN a.gop_indicator = 'RESOLVED' OR a.payment_due_date >= sysdate 
            THEN 0 
            ELSE (sysdate - cast(a.payment_due_date as date)) 
       END                                       as days_overdue       
     , a.sent_date
     , a.reported_date
     , a.resolution_datetime  
FROM FASRPT.GOP_VW   a
     JOIN FASRPT.STUDENT_ENROLLMENT_VW  a1 ON a.enrollment_id = a1.enrollment_id
     JOIN FASRPT.R2T4_CALC_VW b ON a.r2t4_calculation_id = b.r2t4_calculation_id
     JOIN FASRPT.R2T4_CALC_STEP_10_OVERPAY_VW c ON  b.r2t4_calc_step_10_id = c.r2t4_calc_step_10_id
                                                AND a.fund_code = c.fund_code
     LEFT JOIN (SELECT d.enrollment_id, d.fund_code, SUM(d.incremental_amount)  as applied_amount
                FROM FASRPT.CREDIT_BALANCE_ONLY_VW  d
                WHERE d.applied_to_gop = 'True'
                AND   d.credit_balance_rank = 1
                GROUP BY d.enrollment_id, d.fund_code
               ) x  ON  a.enrollment_id = x.enrollment_id
                    AND a.fund_code = x.fund_code
WHERE a.date_of_determination BETWEEN $P{Start_Date} AND $P{End_Date}
AND $X{IN, a.gop_indicator, GOP_Status}
