Student Financial Planning : CONSOLIDATED_PROGRAM_COURSE_VW

Created by Stephen Hutchinson, last modified on Oct 22, 2020

This view is a consolidated view of Student enrollments Program and its associated courses as provided in SAFI. This object stores the information of Unspecified program. The field enrollment_id to be used to link with STUDENT_CONSOLIDATED_VW.
