Student Financial Planning : Data Models

Created by Stephen Hutchinson, last modified by Fernando Medina on Dec 15, 2021

The following data models are deployed in the SFP Oracle BI Publisher instances.

Data Model NameData Model Description
Academic Year OverridesAny academic year overrides took place in prior SAFI along with flag if enrollment is withdrawn and if any pending disbursement.
Application EventApplication events occurring as a part of workflow within certain interval.
At Will LetterProvides an output of the student address and name
Award Letter ReconciliationStudent enrollments whose non-zero award holds are released, disbursements are not cancelled and disbursement status are not 
Disbursed or Disbursement Cancelled or Origination/Disbursement Pending re-opened award year.
Awarding Criteria StatusFunds whose awarding criteria are not fulfilled in academic years  thus pending being awarded.
Awarding ProjectedFunds whose awarding criteria are all fulfilled but fund status is still Projected. This pulls ISIR status along with.
Batch Job ExecutionVarious batch jobs with job name, execution time.
Batch Job ScheduleBatch jobs with their cron schedule.
Batch Job StatisticsBatch jobs with their statistical data like minimum, average, maximum duration.
Batch Job StatusBatch jobs and their run status.
BIP VersionDisplays the current version of deployed SFP Reports, Data Models and Sample Letters
COA Awards ISIRStudent COA information linked to ISIR data via Student Awards.
COD Blocked Transmission ReasonsDisplays various reasons if disbursement status is stuck in Origination/Disbursement ready to be sent to COD.
COD Delayed ResponseCOD messages whose response file yet to receive.
COD ExceptionsCOD messages whose disbursement status is stuck in Origination/Disbursement ready to be sent to COD.
COD Import FailureCOD messages whose disbursement status is stuck in Origination/Disbursement ready to be sent to COD, but file status is Sent 
and student is not in any record hold.
COD Import TransmissionCOD messages whose status is not sent.
COD Student SAIG ErrorOutstanding COD Rejects and affected disbursements.
Common Line Common ResponseCommon response for Commonline funds. It stores information for commonline important data, error handling.
Common Line DiscrepancyCommon response for Commonline funds. It stores information for commonline discrepancies.
Credit BalanceCredit Balance information and its disbursement.
Daily COD RejectsLists rejects from COD
Delayed Package DataStudents that have delayed packages that may require repackaging
Disbursed Funds Without Credit BalanceDisbursed funds with student resource and without any credit balance.
Disbursement Acceptance Information (Letter)List the disbursements (first disbursement from each disbursement group) which are Accepted.
Disbursement Cancel Information (Letter)List funds whose disbursement status is Disbursement Canceled.
Disbursement Criteria FailureDisbursements whose criteria are not fulfilled
Disbursement Information (Letter)List disbursements of Title 4 Grants and Loans sans PLUS funds which are Disbursed.
Document InformationConsolidated information on document request, documents and with their audit info.
FA Cancellation ReconciliationReconcile Ready to be Cancelled holds with cancellation reasons which are Released and has existing Origination.
FAS ProcessDisplay list of Application Processes which Complete or Pending with duration, range.
FISAP DOEData related to the department of education FISAP report
FISAP Other DisbursementData for other disbursements related to the FISAP report
FISAP Part 2Data related to part 2 of the FISAP report
FISAP Part 6Data related to part 6 of the FISAP report
FSEOG AllocationFund amount allocated for FSEOG funds.
Gainful EmploymentInformation of Title IV Direct loans which are Disbursed, along with student, program.
Grant OverpaymentGrant Overpayment and Credit Balance information.
IPEDSProvides student level fund information for the Integrated Postsecondary Education Data System
ISIR Loading IntegrationVarious checkpoints traversed through while loading and matching ISIR
ISIR Verification ResultDisplay of ISIR verification status codes.
Letter HistoryVarious attributes of Common Letters
Manual SAP ReviewSAP information went through manual review.
Money Analytics ReportReports with categories and buckets to fall under for funds which are not disbursed in due course of time.
MPG API ErrorErrors met in MPG
NFF AllocationFund amount allocated for NFF funds.
NSLDS By Packaging VersionHistorical NSLDS information of a student across various packages.
Outstanding FAHEither FA History has not reached school or last received prior to certain number of days.
Packaging Material ChangesTrack the changes in payment period dates, award amounts across the packages if there be any.
Packaging TriggersEvents which triggers packaging along with package versions.
QC Hold System Release CountQC Holds which are released automatically by smart release process.
QC HoldVarious QC holds applied across functional areas.
R2T4 Aid QC HoldsLinking QC holds to R2T4 aided by CARES Act.
R2T4 Process ProgramListing R2T4 with reasons whose status is any of Not Started, Paused or Scheduled.
Return Notification ReconciliationProvides information pertaining to returns related to an R2T4, including QC holds and disbursement status
SAIG Error CODList of SAIG Error with error category of COD.
SAIG Error CPSList of SAIG Error with error category of CPS.
SAIG Error InboundList of SAIG Error with error type of COD_INBOUND_FILE_PROCESS_ERROR
SAIG Error NSLDSList of SAIG Error with error category of NSLDS.
SAIG MailboxList of VUG Audit records.
SAP Academic Information (Letter)List of SAP detailed information.
SAP EvaluationEvaluation matrix of Satisfactory Academic Progress.
Student Award Information (Letter)List of Student COA, Primary EFC, Funds Awarded.
Student Duplicate SSNProvides School student ID and enrollment
Student Enrollment Data DiscrepancyProvides student information when a prior enrollment ID and current enrollment ID match
Student GroupsProvide student level information for student groups
Student MilestoneProgress of students since their initiation in SFP, example student creation date, ISIR received date, NSLDS received date etc.
Student Missing DOBList of students whose DOB is missing in SFP.
Student Notification InformationProvides the student demographic information along with primary program location
Student Record HoldStudents put on hold by school.
Task Statistical InformationList of tasks with start date, end date, date range.
User StatisticsProvides statistical data for users working within task queues 
Valid ISIR Process Status InformationProvides detailed information to populate the Valid ISIR Process Status Information letter
