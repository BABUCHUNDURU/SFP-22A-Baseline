Student Financial Planning : IPEDS Example

Created by Stephen Hutchinson, last modified on Oct 26, 2020

These are examples of queries that reference views in the IPEDS subject.

The code convention $P{INPUT_PARAMETER} is single value input parameter and $X{IN, COLUMN_NAME, INPUT_PARAMETER} is the equivalent of a SQL Language "IN ()" statement. Both of these conventions are used specifically in Jasper Server and map to the input criteria defined within a report definition.

IPEDS Report
SELECT a.school_student_id
     , a.enrollment_id
     , a.program_type
     , a.academic_year_number
     , a.award_year
     , a.fund_code
     , a.fund_name
     , a.fund_type
     , NVL(a.origination_date, a.awarding_date)   action_datetime
     , a.award_amount
     , a.accepted_amount
     , a.grade_level       
     , a.credits 
FROM FASRPT.IPEDS_VW  a
WHERE a.iped_rank = 1
AND    a.award_year = $P{AwardYearIPEDS}
AND    NVL(a.origination_date, a.awarding_date) BETWEEN $P{Start_Date} and $P{End_Date}
