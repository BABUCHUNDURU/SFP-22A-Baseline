Student Financial Planning : PACKAGE_NSLDS_TYPE5_VW

Created by Stephen Hutchinson, last modified on Oct 22, 2020

This view will return FPO data related National Student Load Data System (NSLDS) Type 5 by student enrollment. The field enrollment_id to be used to link with STUDENT_CONSOLIDATED_VW.
