Developer Tools

Supported IDEs:

  • IntelliJ IDEA (recommended tool)
    • Both Community and Enterprise Editions come with built-in Groovy support
    • Configuration instructions for IntelliJ can be found here
  • Eclipse IDE
    • This requires the installation of third-party plugins
    • Instructions for Eclipse can be found here

Limitations

  • Only code completion and syntax checking is supported.
  • Execution / debugging is not supported.
