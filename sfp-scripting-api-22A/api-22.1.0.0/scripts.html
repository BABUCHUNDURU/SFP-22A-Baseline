SFP Configuration Scripts (Release 22.1.0.0)

additionalCoaComponents
Evaluates if a COA configuration is applicable to a student.
awardOverridingCriteria
Override award amounts.
awardYearSelectionCriteria
Determine the award year to use in crossover payment periods.
coaCostValue
Calculates the value of a COA item.
configurableDisbursementCriteria
Evaluates configurable disbursement criteria.
configurableLateDisbursementCriteria
Evaluates configurable late/pwd disbursement criteria.
disbursementDates
Evaluates anticipated disbrusement dates for disbursement.
documentCancellation
Determine when a document request can be canceled.
documentFaAcceptance
Determine the Financial Aid (FA) acceptance of a document.
documentMetadataAdditionalResolutionAction
Evaluate document metadata fields for document acceptance and required supporting documents.
documentOwnerLogic
Determine the document owners.
enrollmentStatusOverride
Groovy script to determine whether the calculated enrollment status (as defined in the School CSV's Enrollment Status Determination column) should be overridden.
exitCounselingGraduatingCriteria
Evaluates exit counseling criteria for graduating students.
exitCounselingWithdrawnCriteria
Evaluates exit counseling criteria for withdrawn students.
globalEnrollmentStatusCriteria
Specifies how the system should determine the student's enrollment status.
gradeLevelProgression
Calculate the grade level.
isirAssumptionAutoCodeClearingLogic
Determine if an ISIR assumption is auto cleared.
isirAssumptionDocumentsRequired
Determine documents required to resolve an ISIR assumption.
isirCCodeAutoCodeClearingLogic
Determine if an ISIR c-code is auto cleared.
isirCCodeDocumentsRequired
Determine documents required to resolve an ISIR c-code.
isirDiscrepancyAdditionalDocumentsRequired
Determine whether additional documentation should be collected for a student in order to ascertain whether an ISIR correction is needed.
isirDiscrepancyEvaluation
Determine whether an ISIR correction for a given ISIR field is required, and if so, what value from what source to correct the ISIR field to.
isirDiscrepancyRequiredDocuments
Determine whether the `isirDiscrepancyEvaluation` script should be executed, based on whether all required documents necessary to make the ISIR correction determination already exist on the student record.
isirHighlightAutoCodeClearingLogic
Determine if an ISIR highlight is auto cleared.
isirHighlightsDocumentsRequired
Determine documents required to resolve an ISIR highlight.
isirRejectCodeAutoCodeClearingLogic
Determine if an ISIR reject code is auto cleared.
isirRejectCodeDocumentsRequired
Determine documents required to resolve an ISIR reject code.
isirUseSubsequentIsir
Define which fields on ISIR will trigger an ISIR review when subsequent ISIRs enter SFP for a Student.
isirVerificationClearanceRules
Determine if an ISIR verification is auto cleared.
isirVerificationDocumentsRequired
Determine documents required to resolve an ISIR verification.
medicalAmount
Determine Medical amount.
nfrAggregateLimit
Evaluates the aggregate limit for an NFR fund.
nfrAnnualLimit
Evaluates the annual limit for an NFR fund.
nfrAwardingCriteria
Evaluates awarding criteria for an NFR period.
nfrDisbursementDates
Evaluates the disbursement date for an NFR period.
nfrEligibility
Evaluates eligibility for an NFR fund.
nfrMaximumProjectedAwardAmount
Evaluates the maximum projected award amount for an NFR period.
nfrMinimumAmount
Evaluates the minimum amount for an NFR fund.
nfrPeriods
Evaluates periods for an NFR fund.
nonTermAcyMonths
Determine the number of months in a non-term academic year.
notificationRequiredCriteria
Determines if a student notification must be triggered when an ISIR is matched to a student or when the student package is updated.
originationCriteria
Evaluates origination criteria for a disbursement.
plusCreditDecisionMatchingCriteria
Determine (G)PLUS loans applicable to the term/payment period being evaluated.
professionalJudgementInitiatingConditions
Determines if a student meets the criteria to initiate a professional judgement process.
professionalJudgementRequiredDocuments
Determines the documents that must be requested for the professional judgement process.
sapDisablePaceOfCompletion
Determines if the Pace Of Completion calculations needs to be considered for the SAP Period evaluation.
sapEvaluationPeriodDetermination
Determines if the SAP Period needs to be evaluated at the end of the academic year.
summerTermType
Defines if the Summer Term should be treated as a Header or Trailer.
termPaymentPeriodMonths
Determine the number of months in a term payment period.
