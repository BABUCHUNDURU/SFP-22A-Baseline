Script: isirDiscrepancyAdditionalDocumentsRequired

Description

Determine whether additional documentation should be collected for a student in order to ascertain whether an ISIR correction is needed. If additional documentation is needed, this script can also trigger document requests for the aforementioned documentation.

Return type
Void
Keywords
isirdiscrepancydocumentadditional-document
Config source
Workbook: ISIR_DISCREP.csv
Field: Additional_Doc_Req_Script
Example
documentRequest.addDocument("Document001", "Student");
